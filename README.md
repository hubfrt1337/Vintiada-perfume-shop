# Vintiada

Vintiada's e-commerce perfume store allows you to purchase items, update pages, and view products in real time.


## Installation

OPEN CMD
git clone https://github.com/hubfrt1337/Vintiada-perfume-shop.git
cd Vintiada-perfume-shop
npm install
npm run server
OPEN SECOND CMD
npm run dev

## Technologies

Vite
React
Express
Cors
